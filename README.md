# Sistema de Orçamento para Serralheria

Projeto desenvolvido em Java com foco em Programação Orientada a Objetos (POO), organização de código, validações de entrada, persistência de dados e experiência do usuário no console.

O sistema permite realizar orçamentos de serviços de serralheria de forma dinâmica, simulando um sistema real utilizado para controle de preços e atendimento de clientes.

---

# Funcionalidades

- Cadastro de serviços
- Cadastro de materiais
- Cadastro e atualização de preços
- Criação de orçamentos
- Cálculo por área ou metro linear
- Sistema de extras no orçamento
- Sistema de desconto em porcentagem ou valor fixo
- Validação de entradas do usuário
- Interface visual aprimorada no console
- Sistema de cores no terminal
- Estrutura orientada a objetos
- Persistência de dados em arquivos `.txt` (em desenvolvimento)
